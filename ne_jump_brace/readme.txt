neに「対応する括弧へのジャンプ」機能を追加するパッチ.

2002/12/12 by Mia(mia@akd.no-ip.org)



1)  概要
    対応する括弧へのジャンプ機能(viの%に相当)をneに追加するパッチ.

2)  機能
    現在のところ,以下の括弧の対応を認識し
    対応するものが見つかればそこへジャンプする.

    /* 1 byte  */ ()<>[]{}
    /* 2 bytes */ (**)/**/‘’“”（）〔〕［］｛｝〈〉《》「」『』【】
    (*と*)や/*と*/は特定言語(笑)のプログラマには便利であろう.

    機能名はsearch_brace_prev及びsearch_brace_nextで
    名前から想像がつくような方向へ対応する文字を検索する.
    デフォルトではそれぞれCtrl-Q[およびCtrl-Q]に割り当てられているが
    キーバインドはneの設定ファイルで変更可能である.

3)  使用方法
    neのソースを展開し、パッチを適用した後neを再コンパイルする.
    本パッチはne000515.tar.gzに対するパッチであるが
    ne-3.05.tar.gzに対しても行オフセットするものの適用可能である.

    $ tar xvfz ne00515.tar.gz
    $ cd ne
    $ patch < /path/to/this/patch
    $ ./configure
    $ make all
    $ make install

5)  BUGS
    現在のところ
        1)  括弧の対応を設定ファイルで変更する手段は提供されていないので
            必要があればソースを修正して再コンパイルする必要がある.
            要望が多ければさらにパッチが提供されることがあるかもしれない.
        2)  括弧を正規表現で定義できない.

    パッチ提出前にテストは行ったが
    発見できなかった不具合が潜んでいるかもしれない.
